# Apple Accessibility Skills

Accessibility audit skills for Apple platforms, designed to be used with AI tools
like Cursor or VS Code agents.

## Platforms
- SwiftUI (iOS + macOS)
- UIKit (iOS)
- AppKit (macOS)

## Purpose
These skills help review UI code with accessibility in mind:
- VoiceOver
- Dynamic Type
- Focus & keyboard navigation
- Contrast and semantics
